## Was Es Ist

`check-for-me.skill` ist ein Risikoprüfer vor dem Senden. Er markiert gefährliche Sätze, erklärt das Risiko und schlägt sicherere Alternativen vor.

## Verwendung

```text
/check-for-me
```
